# Skill - Handoff Writer

## What I do
Estandarizo el traspaso de información entre agentes o sesiones de trabajo, asegurando que el contexto sea mínimo pero suficiente para continuar.

## Structure
- **objetivo completado**: resumen corto.
- **cambios claves**: archivos modificados y por qué.
- **evidencia**: logs, capturas o resultados de tests.
- **riesgos**: deudas técnicas o puntos de atención.
- **siguiente accion**: qué debe hacer el siguiente agente.

## When to use me
Al finalizar una tarea o una sesión de trabajo de un agente.
