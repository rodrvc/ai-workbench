# Policy - Definition of Done

Una tarea se considera terminada cuando:

1. Se cumplio el objetivo del `HU Brief`.
2. El handoff esta completo y accionable.
3. Se registraron riesgos y pendientes.
4. Existe evidencia minima (tests/logs/verificacion).
5. Se actualizo aprendizaje en knowledge/postmortem.
