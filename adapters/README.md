# Adapters

Carpeta para traducciones ligeras entre runtimes, sin tocar el core.

## Regla

- El core permanece neutral.
- Cada adapter solo mapea inputs/outputs al `Interop Contract`.

## Ejemplos de adapters

- `claude/`
- `gpt/`
- `gemini/`
